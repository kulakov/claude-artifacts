# Пример рабочего процесса PM

Представьте себе типичный рабочий процесс продакт-менеджера:

**Утро:**
- Попросите Claude резюмировать вчерашние заметки со встречи
- Наблюдайте, как файл с резюме появляется в Obsidian
- Прочитайте, внесите правки при необходимости
- Поделитесь с командой

**День:**
- Попросите Claude составить черновик PRD на основе пользовательских исследований
- Видите, как PRD появляется в реальном времени
- Просматривайте его в Obsidian, общаясь с Claude
- Запросите правки: "Сделай раздел 3 более техническим"
- Наблюдайте, как изменения происходят вживую

**Конец дня:**
- Попросите Claude обработать все задачи за сегодня
- Увидите, как появляется организованный список
- Будете точно знать, что делать завтра

Это ваша новая реальность! Вы работаете ВМЕСТЕ с Claude, видя всё, всегда контролируя процесс.
