# Claude Artifacts

Хостинг для артефактов, созданных в Claude.

## Как добавить артефакт

1. Сохрани HTML/React артефакт из Claude в папку `/artifacts/`
2. Закоммить и запушить
3. Артефакт будет доступен по адресу:
   ```
   https://kulakov.github.io/claude-artifacts/artifacts/имя-файла.html
   ```

## Структура

```
/artifacts/          — готовые артефакты
/templates/          — шаблоны для разных типов
```

## Быстрый деплой из Claude

Скачай артефакт → положи в `/artifacts/` → push
